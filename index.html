云计算技术 课程实验报告

学号：202002130245 姓名： 吴祥钧 班级： 20数据
实验题目：利用云平台搭建个人博客
实验学时：2 实验日期：2023/3/14

实验目的：熟悉个人博客系统的搭建。

具体包括：

参考方案：注册Github账号，搭建Hexo环境并实现个人博客搭建，撰写实验报告。

硬件环境： 

联网的计算机一台

软件环境：

Windows or Linux

实验步骤与内容：

首先下载node.js和git，然后使用npm安装Hexo，输入以下命令npm install hexo-cli -g。

然后创建Blog，npm install -g hexo-cli

初始化 hexo init myblog

然后在github上创建仓库

生成SSH添加到GitHub

然后把生成的id\_rsa.pub里面的信息复制进去。

打开站点配置文件 \_config.yml，翻到最后，修改为你的GitHub账户

需要先安装deploy-git ，也就是部署的命令,这样你才能用命令部署到GitHub。

npm install hexo-deployer-git --save

然后执行

hexo clean

hexo generate

hexo deploy

就能成功搭建个人博客了

发布文章进入博客所在目录，右键打开 Git Bash Here，创建博文：

hexo new "My New Post"

然后 source 文件夹中会出现一个 My New Post.md 文件，就可以使用 Markdown 编辑器在该文件中撰写文章了。

写完后运行下面代码将文章渲染并部署到 GitHub Pages 上完成发布。以后每次发布文章都是这两条命令。

hexo g # 生成页面

hexo d # 部署发布

| |

结论分析与体会：

学会了使用 Hexo 创建静态博客网站，

学会了使用 Git 进行版本控制，并将自己的博客网站托管到 GitHub Pages 上。

在本实验中，我们使用了Git将我们的博客托管在GitHub上，可以使博客网站非常容易地进行版本控制和部署。每次你更新博客文章后，只需要在命令行中输入一条命令，就可以将新的内容部署到 GitHub Pages 上。
